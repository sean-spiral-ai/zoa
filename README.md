# Baseline Go Coding Agent

A minimal but practical coding agent in Go with Gemini function-calling and a baseline coding toolset inspired by Pi's core tools.

## What is included

- Clean LLM API boundary in `internal/llm`:
  - `types.go`: model-agnostic request/response/message types.
  - `gemini.go`: Gemini GenerateContent client + function-calling mapping.
- Agent session loop in `internal/agent/session.go`:
  - iterative model/tool turns
  - grouped tool responses per model turn
  - max-turn + timeout controls
- Baseline coding tools in `internal/tools`:
  - `read`, `write`, `edit`, `bash`, `ls`, `find`, `grep`
  - workspace-root path enforcement
  - bounded output truncation helpers
- Simple CLI in `cmd/coding-agent/main.go`.

No plugin system and no fancy terminal UI are included.

## Model used

Development and verification runs used:

- `gemini-3-flash-preview`

## Usage

```bash
go run ./cmd/coding-agent \
  --model gemini-3-flash-preview \
  --cwd /absolute/workspace/path \
  --prompt "Create a small project and run tests" \
  --verbose
```

Required env var:

- `GEMINI_API_KEY`

## Tool behavior highlights

- `read`: line-offset reading with truncation notices for large files.
- `write`: creates parent dirs and overwrites target file.
- `edit`: exact + unique snippet replacement.
- `bash`: executes via `bash -lc` in workspace root; supports timeout.
- `find`: glob search with `**` support; ignores `.git` and `node_modules`.
- `grep`: content search (literal/regex), optional context + glob filter.
- `ls`: sorted directory listing with directory suffixes.
